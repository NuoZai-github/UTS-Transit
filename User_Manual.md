## 1. Introduction
**UTS Transit** is a smart campus mobility solution designed to make transportation at University of Technology Sarawak (UTS) more predictable and efficient.

## 2. Getting Started

### System Requirements
*   **Mobile**: Android 8.0+ / iOS 14+
*   **Web**: Chrome / Edge / Safari

### Installation
1.  **Android**: Install the provided `UTSTransit.apk`.
2.  **Web**: Access via `https://web-admin-nuozai.vercel.app`.

## 3. Registration Guide (Important)

### Student Registration
Most users will register as students.
1.  Launch the App.
2.  Tap **Sign up here**.
3.  Enter **Email**, **Password**, **Full Name**, **Student ID**.
4.  Role: Select **"Student"**.
5.  Tap **Register**.

### Driver & Admin Registration (Restricted)
To prevent unauthorized access, registering as a **Driver** or **Admin** requires a security code.

1.  On the Sign Up page, enter your details.
2.  Select Role: **"Driver"** or **"Admin"**.
3.  A **"System Code"** field will appear (or you will be prompted).
4.  Enter the security code: 
    > **Code: `LIME6199`**
5.  Tap **Register**. 
    *   *Note: If the code is incorrect, the system will revert your role to 'Student' or block registration.*

## 4. Student Features

### Booking Trips
1.  **View Schedule**: Check the Home screen for today's trips.
2.  **Book**: Tap "Book Trip". The status changes to "Booked".
3.  **Cancel**: Tap again to cancel (if supported).

### Live Tracking
1.  Go to **Map** tab.
2.  **Allow Location Permissions** (Select "While using the app").
3.  Watch the **Bus Icon** ðŸšŒ move in real-time.
4.  Check the bottom **Status Bar** for trip progress.

---

## 5. Driver Features

### Starting a Journey
1.  Log in as Driver.
2.  Select today's trip from the dropdown.
3.  Check passenger count.
4.  Tap **Green "Start Journey"**.
    *   **Alert**: You must grant location permissions.
5.  Status becomes **Driving**.

### Ending a Journey
1.  Tap **Red "End Journey"** upon arrival.

---

## 6. Admin Web Portal

1.  Login at the Web URL using Admin credentials.
2.  **Dashboard**: Monitor active buses and student statistics.
3.  **Tracking**: View real-time map of all buses.
4.  **Announcements**: Post notifications for the mobile app.

---

## 7. Troubleshooting

| Issue | Solution |
|-------|----------|
| **Registration Failed** | Check if email is valid. If Driver, ensure Code `LIME6199` is used. |
| **Location Error** | Go to Settings > Apps > UTSTransit > Permissions > Location > Allow. |
| **No Bus on Map** | Bus may not have started yet. Wait for "Driving" status. |
